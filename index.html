Project Overview

HealthGen is a cross-platform mobile application built with Flutter. It provides users with personalized health-related services, including health tips, symptom assessment, medicine reminders, medical service locators, user management, and notification handling.

The application is structured using a robust and scalable architecture based on feature-wise module separation, provider/state management, and clean folder organization.
